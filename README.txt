Инструкция по применению: 

		Это калькулятор. Он имеет функцию суммы и разности,
		оснащен автоматизированной сборкой.
		
	Необходимо перейти в консоль и прописать:
		$ ant run
	Чтобы запустить тесты прописываем команду в консоли: 
		$ ant test
	Удаление каталога build:
		$ ant clear
	Чтобы скомпилировать проект, прописываем:
		$ build
